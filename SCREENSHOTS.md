## User Flow

1. **App Launch**: Opens to map centered on San Francisco
2. **Exploring**: User can pan and zoom to explore different areas
3. **Adding Bins**: Tap location to mark new garbage bin
4. **Managing View**: Use zoom controls or reset button
5. **Clearing**: Reset to default state with clear button
